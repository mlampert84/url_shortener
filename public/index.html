URL Shortener Microservice

Created according to the freeCodeCamp backend API Project URL Shortener Microservice.

Usage

  • Pass a valid URL parameter of the form /new/<valid_url>. For example: http://thawing-mesa-25685.herokuapp.com/new/http://google.com
  • Based on the url parameter you entered as /new/<valid_url>, this microservice returns a JSON object containing the valid url you entered, "original_url", as well a "short_url". For example: {"original_url":"http://google.com","short_url":"thawing-mesa-25685.herokuapp.com/353"}. If the original url already exists in the database, then it returns that entry. If the url doesn't already exist in the database, a new database entry is created.
  • When you enter the "short_url" into your browser window, you will be redirected to the "original_url". For example: thawing-mesa-25685.herokuapp.com/353 would redirect, in this example, to http://google.com.
